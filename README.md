# Laravel Contact Form

このプロジェクトは Laravel を使ったお問い合わせフォームの練習用アプリです。Docker を使ってローカル環境を構築し、MySQL データベースと連携しています。

## 📦 使用技術

- Laravel 10
- PHP 8.x
- Docker / Docker Compose
- MySQL
- Bladeテンプレート
